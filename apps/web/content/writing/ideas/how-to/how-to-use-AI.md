# How To Use AI

这个主要目的是一个头脑风暴，就是分析对不同的人如何方便的使用AI工具解决实际的问题，核心问题是
如何更方便的使用解决问题的这个路径，来达到一定的客户粘度。
客户可以分几类：
1. 完全小白，但是听过AI，有基本素质，比如本科教育，但是完全没有计算机基础
2. 完全小白，电脑使用也是一点点，但是手机使用完全没有问题
3. 有一定的专业背景，但是不是计算机，使用过AI工具，但是只是chatbox，有很多怀疑，因为用的可能是豆包，元宝等等
4. 有非常强的专业背景，但是不是计算机，希望使用AI工具帮助
5. 有计算机背景，但是没有业务背景，希望使用AI工具可以提高自己技能，找到可以落地的场景

以上是主要的自己分析的可能客户群体，请根据不同的客户群里进行分析，分析出每个客户群的特征，给出不同类别的用户如何使用AI工具的可能场景，如果入门，投入，然后更好的利用AI工具。

人群分析只是我拍脑袋想的，如果可以更具体话，你可以按照象限这种方式分析，可能每一个象限里面会有不同分布情况。
