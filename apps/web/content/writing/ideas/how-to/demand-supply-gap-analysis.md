### 2.3 情绪型内容 vs 分析型内容的比例

虽然未找到精确的量化比例数据，但多来源的证据一致表明：

| 维度 | 结论 |
|------|------|
| 短视频平台（抖音/快手） | 情绪驱动内容（鸡汤/站队/毒鸡汤）**压倒性占多数** |

## 第四部分：风险与反驳

### 4.1 可能反驳"缺口不大"的论点及回应

| 反驳 | 回应 |
|------|------|
| "心理学已经覆盖了这些话题" | 心理学覆盖了话题但没有覆盖框架。用户听到"你是焦虑型依恋"和"你的大脑在执行一个概率更新程序"是两种完全不同的体验。心理学给标签，概率论给理解。 |
| "薛兆丰/李永乐已经在做" | 薛兆丰是经济学框架且定位学术/课程；李永乐只做了1-2条恋爱视频，不是系列。没有人系统化。 |
| "这种内容太冷/没人看" | 李永乐单条百万播放、中二所直播144万在线、曹老师250万点赞——数学x恋爱的传播力已经被反复验证。 |
| "中文读者不需要数学框架" | Soul/F复日报告：77.3%的年轻人主动用分析框架看恋爱。"恋爱做题家"已成为社会现象。 |
| "小红书上做不了这种内容" | 小红书情感赛道已验证流量池；数英网分析"赛道越小众越吃香"；"人间清醒"标签正是理性分析的入口。 |

### 4.2 主要风险

1. **内容门槛**：概率论/贝叶斯需要准确且通俗的表达，质量要求高
2. **模仿风险**：先发后可能被快速模仿——内容深度和速度是壁垒
3. **受众限制**：部分用户可能对数学有本能排斥——类比和故事化是关键
4. **平台算法**：小红书/抖音算法偏好情绪化内容——需要测试理性内容是否能获得分发
