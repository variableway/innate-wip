# "概率论/贝叶斯/博弈论 x 恋爱/两性关系" 竞争格局分析报告

> 调研时间：2026-04-17 ~ 2026-04-20
> 调研目的：验证"概率论/贝叶斯/博弈论 x 恋爱"是否为中文市场真实空白
> 调研方法：多轮交叉搜索（30+ 组关键词），覆盖知乎、B站、小红书、豆瓣、微信读书、智慧树、极客时间等平台

---

## 核心结论（Executive Summary）

**"概率论/贝叶斯 x 恋爱"是一个真实的、未填补的内容空白。**

"博弈论 x 恋爱"和"经济学 x 恋爱"已有零散内容，但均未系统化、未IP化、未形成系列。**没有任何一个中文创作者或内容IP在系统性使用概率论/贝叶斯框架解释恋爱关系。**

关键发现：
1. 概率论/贝叶斯框架在中文恋爱内容中几乎为零——这是最大的空白
2. 博弈论框架有零散内容（知乎文章、B站视频），但没有系列化内容IP
3. 经济学框架已有较成熟内容（课程、书籍），但定位偏学术/教材，不是社交媒体内容
4. 小红书上几乎没有人用数学/概率框架分析恋爱——这是核心目标平台的真空地带

### 9.3 风险与注意事项

3. **KnowYourself可能尝试**——他们团队大，但一直坚持心理学框架。风险低。
