# Word Report Tool


## Task 1: 调研word工具场景

解决编写报告的问题，如何使用AI工具。场景如下：
1. 比如有一份word格式文档，然后有一份有部分接近的文档，通过程序实现自动填充报告内容，尽可能少改动
2. 一份只有格式的word格式文档，和一份主要观点的描述

一下是一些方向需要调研和考虑:

1. 使用当前AI工具调用，kimi，豆包等，是什么的内容可以完全生成，格式定义需要的prompt是什么？
2. 如果这些生成需要什么设定可以满足要求
3. 如果不行是否可以通过wps ai功能
4. 是否可以通过飞书文档功能？
5. 通过自定义插件功能，比如word插件，实现自动填充报告内容，比如标题，段落，表格等。
6. 比如开源Office工具
7. 直接程序编写，比如有一份work格式文档，然后有一份有部分接近的文档，通过程序实现自动填充报告内容，尽可能少改动
8. 是否可以使用chrome pluging工具，可以吧对话内容直接复制到word文档而不需要调整格式

同时给出一份有5-10个AI word工具的分析，如果是github上开源的最好。
