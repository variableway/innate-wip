# Making

Core project tracking dashboard at `/making`.

## Sub-sections

### Issues (`/making/issues`)
Browse and filter GitHub issues from all tracked repositories. Features status filtering, project filtering, and RSS feed.

### Projects (`/making/projects`)
Project showcase with AGENTS.md documentation, AI-generated analysis (features, strengths, weaknesses), and external repo links.

### Weekly (`/making/weekly`)
Weekly progress summaries with AI-generated evaluations and mindset analysis. Supports bilingual content (Chinese & English).

### Insights (`/making/insights`)
AI-generated insights derived from project activity and weekly reports.

## Data Source
- `apps/web/data/issues.json` — GitHub issues
- `apps/web/data/projects.json` — Project analysis
- `apps/web/data/weekly.json` — Weekly summaries
- `apps/web/data/insights.json` — AI insights
