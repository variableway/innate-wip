# Writing

Blog platform for thoughts, ideas, and experiences at `/writing`.

## Features
- **Post listing** — Published posts with metadata
- **Table of contents** — Sidebar navigation within posts
- **RSS feed** — Available at `/feed/rss.xml`
- **Markdown rendering** — Full markdown support with syntax highlighting

## Data Source
- `apps/web/content/posts/` — Markdown/MDX files with frontmatter

## Key Files
- `apps/web/lib/content/` — Content loading and processing
- `apps/web/app/writing/` — Writing pages and routing
