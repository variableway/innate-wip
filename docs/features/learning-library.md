# Learning Library

Course catalog for educational resources at `/learning-library`.

## Features
- **Course grid** — Grid layout displaying available courses
- **Course detail** — Individual course pages with sections and lessons
- **Progress tracking** — Course completion state

## Data Source
- `apps/web/lib/data.ts` — Course data

## Key Files
- `apps/web/app/learning-library/` — Library pages
- `apps/web/app/course/[id]/` — Individual course pages
