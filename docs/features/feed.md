# Feed

Content discovery feed at `/feed`.

## Features
- **Aggregated feed** — Shows stories and expertise from various writers
- **60-second ISR** — Incremental Static Regeneration for fresh content
- **Engagement metrics** — Likes, comments, and read time
- **Filtering** — By category (article, log, news) and editor's picks

## Data Source
- Published posts metadata converted to feed format

## Key Files
- `apps/web/app/feed/` — Feed pages
