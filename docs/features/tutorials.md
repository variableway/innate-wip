# Tutorials

5-minute quick-start tutorials at `/tutorials`.

## Features
- **Quick tutorials** — Copy-and-modify examples with immediate results
- **Difficulty levels** — Grouped by experience level
- **Go language content** — Includes Go learning materials

## Data Source
- `apps/web/lib/tutorials-server.ts` — Server-side content loading
- `apps/web/lib/tutorials.ts` — Tutorial data types

## Key Files
- `apps/web/app/tutorials/` — Tutorial pages and routing
- `apps/web/components/tutorials/` — Tutorial-specific components
